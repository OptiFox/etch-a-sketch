# The Odin Project: Etch-a-Sketch

## Project Overview

This repository contains my solution for the Etch-a-Sketch project by The Odin Project, which is the toughest one yet for me. I have learned quite a lot from this one, such as how to properly make a grid using CSS, randomizing colors, opacity increment, using JavaScript functions like prompt(), confirm(), etc.

## Features

This page contains the following:

* **Grid:** This is where you can draw in this project.
* **Configure Grid Button:** You can configure the size of the grid from here by inputting a number, and it will make a grid based on that. (Input: 16, Grid: 16x16)
* **Clear Grid Button:** You can clear the grid by pressing this button.
* **Reset Grid Button:** You will be asked for confirmation before resetting to default grid size (16x16).

## Technologies Used

* **HTML5:** For the main page layout.
* **CSS3:** For the styling and coloring.
* **JavaScript:** For an interactive web experience.
